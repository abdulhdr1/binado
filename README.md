# Download the script
```
wget https://raw.githubusercontent.com/abdulhdr1/binado/refs/heads/main/binado
```

# Make the script executable
```
chmod +x binado
```

# Move the script to a directory in your PATH (e.g., ~/.local/bin)
```
mv binado ~/.local/bin/
```

# Usage
```
binado /path/to/file.desktop
```
